# 💬 ShhTalk – Real-Time Chat Application

A real-time chat application built as part of my internship project using **Node.js**, **Express**, **Socket.IO**, and **MongoDB**. It includes modern features such as:

- 🎨 Dark mode
- 🔤 Typing indicators
- 😊 Emoji support
- 📡 Real-time messaging
- 🧠 User authentication
- 💾 MongoDB database integration

## 🧪 Getting Started

 **1. Clone the repository**
git clone https://github.com/yourusername/ShhTalk.git
cd ShhTalk

**2. Install dependencies**
npm install

**3. Set up your environment variables2.**
Create a .env file in the root directory based on .env.example:

PORT=5000

MONGO_URI=your_mongo_db_connection_string

JWT_SECRET=your_jwt_secret

**4. Run the server**
node server.js
Then open http://localhost:5000 in your browser to use the app.
